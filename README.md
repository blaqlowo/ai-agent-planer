AI Agent Orchestrator is a multi-step reasoning system designed to simulate autonomous task execution.

Given a high-level objective, the system:
1. Interprets intent  
2. Breaks the task into structured steps  
3. Executes each step using AI  
4. Maintains context and memory  
5. Produces a final, refined output  

---

## ⚙️ Core Features

- Goal → Plan → Execute architecture  
- Multi-step reasoning loop  
- Context-aware memory handling  
- Execution logging for transparency  
- Modular agent design
